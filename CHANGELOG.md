# Changelog

このプロジェクトの主な変更履歴を記録します。

## [1.3.1] - 2026-06-14

### Changed

- READMEに元記事へのクレジットを追加しました。
- NOTICE.mdを追加し、元コードの参考元と本アプリでの改修内容を明記しました。
- LICENSEの著作権表記を `ShinonomeSyusui and contributors` に更新しました。
- リリース表記を v1.3.1 に更新しました。

### Fixed

- 最前面表示ON時に透明度設定画面が背面へ回る問題を修正しました。

## [1.3.0] - 2026-06-14

### Added

- 月日＋曜日の日付表示オプションを追加しました。
- 透明度を50〜100%のスライダーで調整できる設定画面を追加しました。
- Pillowを使った高画質描画を追加しました。
- LED発光効果のON/OFFを追加しました。
- 高画質描画とLED発光効果を表示メニュー、右クリックメニュー、設定保存に対応しました。
- LED発光効果が高画質描画OFFでも反映されるようにしました。
- 高画質描画の縮小処理をくっきり見える方向へ調整しました。
- ユーザー指定の `assets/app_icon.png` を加工せず、アプリアイコンとして使う方式に戻しました。
- ウィンドウとタスクバーのアイコン読み込みを、公開版で使用していた画像に合わせて調整しました。
- WindowsのDPI認識とTkメニューフォント設定を追加し、メニューバー文字を読みやすくしました。
- LED発光後に数字本体を描き直し、発光ON時も輪郭が滲みにくいようにしました。
- LED発光ON/OFFで日付サイズが変わらないよう、日付レイアウト計算を共通化しました。
- 最小サイズ時に時計と日付が重ならないよう、日付領域と間隔を調整しました。
- 日付の7セグ数字と年月日ラベルのサイズバランスを調整しました。

## [1.2.1] - 2026-06-13

### Fixed

- 日付表示ありの通常表示で、メニューバー直下の時計上部余白を調整しました。
- 追加余白ぶんを時計表示領域から差し引き、時計と日付が重ならないようにしました。

- 最前面表示ON/OFF
- 秒表示ON/OFF
- 24時間表示 / 12時間表示切り替え
- 秒サイズの標準 / 小さめ切り替え
- 時計のみ表示
- 時計のみ表示中の右クリックメニュー
- ウィンドウサイズと位置の保存
- アプリアイコン設定
- 日付表示
- 曜日色表示
- テーマプリセット
- 自作テーマ1〜3の保存と読み込み
- レイアウトプリセット
- Windows起動時の自動起動
- 画面外に出たウィンドウの復帰
- 設定のエクスポート / インポート
- ショートカットキー
