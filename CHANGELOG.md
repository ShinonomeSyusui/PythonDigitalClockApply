# Changelog

このプロジェクトの主な変更履歴を記録します。

## [1.4.4] - 2026-06-17

### Changed

- 時計の更新タイミングを、固定間隔ではなくPC時刻の秒境界に合わせて補正するように変更しました。
- 秒表示ON時の更新処理が、長時間動作時にもPC時刻へ自然に再同期しやすくなるように改善しました。
- 秒表示OFF時のコロン点滅についても、500ms間隔を基準にしつつ、現在時刻に合わせて次回更新タイミングを計算するように調整しました。

### Fixed

- after(1000) 固定更新によって、処理遅延や長時間起動時に表示更新タイミングが少しずつずれる可能性を軽減しました。
- 長時間起動時や処理遅延後でも、時計表示がPC時刻からずれにくくなるように改善しました。

## [1.4.3] - 2026-06-16

### Added

- 背景を透明化できる「背景を透明にする」設定を追加
- 背景透明化ON時に、数字をくっきり表示したまま背景だけを透明化する機能を追加
- 背景透明化ON時に、セグメント消灯部分も透明化するように対応
- 背景透明化ONかつ時計のみ表示モード中でも、透明部分で右クリック・ドラッグ操作できる操作用レイヤーを追加

### Changed

- 既存のウィンドウ全体透明度とは別機能として、背景のみ透明化できるように変更
- 背景透明化ON/OFFを settings.json に保存し、次回起動時にも反映されるように変更

### Safety

- Windows以外、または背景透明化が利用できない環境でもアプリが落ちないように安全対策を追加

## [1.4.2] - 2026-06-16

### Changed

- 秒表示OFF時に、コロンの点滅を500ms間隔で行うように変更

## [1.4.1] - 2026-06-15

### Added

- 閉じるボタン押下時に、アプリを終了せずタスクトレイへ格納できる設定を追加
- 設定メニューに「閉じるボタンでタスクトレイに格納」を追加
- 設定内容を settings.json に保存し、次回起動時にも反映されるように対応

### Changed

- 設定OFF時は従来通り、閉じるボタンで完全終了する動作を維持
- 設定ON時は、閉じるボタンでウィンドウを非表示にし、タスクトレイから復帰できるように変更
- トレイメニューの「終了」は、設定状態に関係なく常に完全終了する動作を維持

### Safety

- タスクトレイが利用できない環境では、ウィンドウを非表示にせず従来通り完全終了するようにし、復帰不能になる事故を防止


## [1.4.0] - 2026-06-15

### Added

## [1.3.0] - 2026-06-14

### Added

- 月日＋曜日の日付表示オプションを追加しました。
- 透明度を50〜100%のスライダーで調整できる設定画面を追加しました。
- Pillowを使った高画質描画を追加しました。
- LED発光効果のON/OFFを追加しました。
- 高画質描画とLED発光効果を表示メニュー、右クリックメニュー、設定保存に対応しました。
- LED発光効果が高画質描画OFFでも反映されるようにしました。
- 高画質描画の縮小処理をくっきり見える方向へ調整しました。
- ユーザー指定の `assets/app_icon.png` を加工せず、アプリアイコンとして使う方式に戻しました。
- ウィンドウとタスクバーのアイコン読み込みを、公開版で使用していた画像に合わせて調整しました。
- WindowsのDPI認識とTkメニューフォント設定を追加し、メニューバー文字を読みやすくしました。
- LED発光後に数字本体を描き直し、発光ON時も輪郭が滲みにくいようにしました。
- LED発光ON/OFFで日付サイズが変わらないよう、日付レイアウト計算を共通化しました。
- 最小サイズ時に時計と日付が重ならないよう、日付領域と間隔を調整しました。
- 日付の7セグ数字と年月日ラベルのサイズバランスを調整しました。
