# NOTICE

PythonDigitalClockApply is a desktop digital clock application developed and maintained by ShinonomeSyusui.

This project was created as an experiment to improve and package a Python/Tkinter seven-segment digital clock sample into a more complete Windows desktop application.

## Original reference

The initial seven-segment clock sample code was based on the following article:

* Site: だえうホームページ
* Article: 「【Python/tkinter】デジタル時計の作り方（７セグ表示版）」
* URL: https://daeudaeu.com/tkinter-digital-clock/

The original article and sample code are credited to their original author/site.

## Modifications

Major modifications and packaging work by ShinonomeSyusui include:

* Application structure refactoring
* Responsive window resizing
* Segment color customization UI
* Settings persistence
* Date display options
* Window opacity settings
* High-quality rendering support
* LED glow effect
* Windows packaging
* GitHub Release distribution
