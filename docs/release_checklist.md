# リリースチェックリスト

## 1. リリース前

- [ ] アプリケーションが正しく起動することを確認する
- [ ] 設定が正しく保存および読み込まれることを確認する
- [ ] アイコンが正しく表示されることを確認する
- [ ] README と CHANGELOG が更新されていることを確認する
- [ ] バージョン番号が更新されていることを確認する

## 2. ビルド

- [ ] build.bat を実行する
- [ ] build_release.bat を実行する
- [ ] dist フォルダに実行ファイルが生成されることを確認する
- [ ] release フォルダに ZIP ファイルが生成されることを確認する
- [ ] ZIP ファイルを解凍して、実行ファイルが正しく起動することを確認する
- [ ] 実行ファイルが生成されることを確認する
- [ ] 実行ファイルが正しく起動することを確認する
- [ ] デスクトップショートカットアイコンが正しく表示されることを確認する

## 3. GitHub

- [ ] プルリクエストを作成する
- [ ] main にマージする
- [ ] 最新の main をローカルにプルする
- [ ] GitHub で新しいリリースを作成する
- [ ] リリースアセットをアップロードする
- [ ] ダウンロードリンクが機能することを確認する
