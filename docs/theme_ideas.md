# v1.5.0 テーマ追加案

## 概要

v1.5.0 では、SevenSegmentClock のテーマプリセット拡充を検討する。

v1.4.5 でショートカット操作や背景透明化時の挙動が安定したため、次の改善として見た目の選択肢を増やし、ユーザーが好みに合わせて時計の雰囲気を変えやすくする。

## 方針

* v1.4.5 の安定性を崩さない
* 時計更新処理、背景透明化処理、タスクトレイ処理には大きく手を入れない
* まずは固定テーマの追加を中心に進める
* 既存テーマと印象が被りすぎないようにする
* 長時間表示しても目が疲れにくい配色を優先する
* 背景透明化ON時でも見やすいか確認する

## 採用基準

* 背景色と点灯色のコントラストが十分あること
* セグメント消灯色が目立ちすぎないこと
* 既存テーマと雰囲気が被りすぎないこと
* 通常表示で見やすいこと
* 背景透明化ON時でも大きく見づらくならないこと
* 長時間表示しても目が疲れにくいこと

## 作業メモ

* まずは配色案を検討する
* 良さそうなテーマだけ実装候補に残す
* 一度に大量追加せず、品質の良いテーマを優先する
* README にテーマ一覧を追記するか検討する
* CHANGELOG に v1.5.0 の変更内容を記録する
